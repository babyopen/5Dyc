# 🔮 六合彩生肖预测系统

基于 XGBoost 集成学习的六合彩生肖预测系统，支持从接口获取最新数据并进行智能预测。

## 📁 项目结构

```
稳定版前5对/
├── scripts/              # 主要预测脚本
│   ├── predict_next.py           # 完整预测系统（使用多年历史数据）
│   ├── predict_2026_only.py      # 2026年专用预测
│   └── predict_2025_2026.py      # 2025-2026年预测
│
├── utils/                # 工具脚本
│   ├── add_2026106.py            # 数据添加工具（支持命令行参数、验证、备份）
│   ├── check_2026_data.py        # 数据完整性检查
│   ├── update_and_check_data.py  # 数据更新和检查
│   ├── query_period.py           # 期号查询工具
│   ├── filter_data.py            # 数据过滤工具
│   ├── add_latest_period.py      # 添加最新期数
│   ├── update_2026_latest.py     # 更新2026最新数据
│   └── update_special_number.py  # 更新特码
│
├── data/                 # 数据文件
│   ├── lottery_data.json         # 主数据文件（106期）
│   ├── lottery_data_filtered.json # 过滤后的数据
│   └── *.backup_*                # 自动备份文件
│
├── models/               # 模型文件目录
├── docs/                 # 文档目录
├── old_version/          # 旧版本模型和脚本
├── .gitignore            # Git忽略配置
└── README.md             # 本文件
```

## 🚀 快速开始

### 1. 环境要求

```bash
pip install pandas numpy requests xgboost scikit-learn
```

### 2. 预测下一期

#### 方式一：使用完整历史数据（推荐）
```bash
cd scripts
python3 predict_next.py
```

**特点：**
- 使用 2020-2026 年共 2560+ 期数据
- 更高的预测准确性
- 训练时间稍长

#### 方式二：仅使用2026年数据
```bash
cd scripts
python3 predict_2026_only.py
```

**特点：**
- 仅使用 2026 年 106 期数据
- 训练速度快
- 适合近期趋势分析

### 3. 添加新期数数据

```bash
cd utils
python3 add_2026106.py -p 2026107 -s 15 -z 3 -n 5 12 23 34 41 48 -y
```

**参数说明：**
- `-p`: 期号（7位数字）
- `-s`: 特码（1-49）
- `-z`: 生肖代码（1-12）
- `-n`: 6个平码
- `-t`: 开奖时间（可选）
- `-y`: 跳过确认

**生肖代码对照：**
```
1=马, 2=蛇, 3=龙, 4=兔, 5=虎, 6=牛
7=鼠, 8=猪, 9=狗, 10=鸡, 11=猴, 12=羊
```

### 4. 检查数据完整性

```bash
cd utils
python3 check_2026_data.py
```

### 5. 从接口更新数据

```bash
cd utils
python3 update_and_check_data.py
```
